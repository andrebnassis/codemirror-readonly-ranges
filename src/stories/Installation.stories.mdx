## Installation
https://www.npmjs.com/package/codemirror-readonly-ranges

```bash
# Using npm
npm install codemirror-readonly-ranges range-analyzer @codemirror/state @codemirror/view
```

```bash
# Using yarn
yarn add codemirror-readonly-ranges range-analyzer @codemirror/state @codemirror/view
```

PS: Obligatory install dependencies:
```json
"range-analyzer"
"@codemirror/state"
"@codemirror/view"
```
